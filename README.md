# MarbleClan: Tales of the Knoxville Hollow

A short illustrated novella for Eli (10) and Rosie (8) — a Warrior Cats–style story about a clan of cats living in modern Knoxville who learn the city's history through the tales an elder tells. Built April 2026 ahead of the family trip.

## Hosting on GitHub Pages

1. Create a new public repo on GitHub (e.g. `marbleclan-knoxville`).
2. From this folder:

   ```bash
   git remote add origin git@github.com:<your-username>/<repo-name>.git
   git branch -M main
   git push -u origin main
   ```

3. On GitHub: **Settings → Pages → Source: Deploy from a branch → Branch: `main` / `(root)` → Save**.
4. Within ~1 minute the book will be live at `https://<your-username>.github.io/<repo-name>/`.
5. Open that URL on the iPad.

## Local viewing (no GitHub needed)

Just double-click `index.html` to open in any browser. Everything is bundled — no build step.

## Files

- `index.html` — the book itself (single-file HTML + CSS).
- `images/` — nine illustrations (cover + 8 chapters), generated with Google Nano Banana in James L. Barry / Warrior Cats graphic novel style.
- `prose.md` — the source manuscript in plain markdown, in case you want to edit prose without touching HTML.

## Contents

- Prologue: The Hollow
- Ch 1: The Cat Who Built the Walls *(James White's fort, 1786)*
- Ch 2: The Cat Who Made Words *(Sequoyah's syllabary)*
- Ch 3: The Mountain That Breathes *(Great Smokies, synchronous fireflies)*
- Ch 4: The Secret City *(Oak Ridge / Manhattan Project)*
- Ch 5: The Sea Beneath *(Lost Sea, prehistoric jaguar)*
- Ch 6: The Year the Whole World Came *(1982 World's Fair / Sunsphere)*
- Epilogue: The Voice in the Theatre *(Tennessee Theatre, Wrath of Khan + Shatner)*

Every history fact is verifiable. Every cat is invented.
